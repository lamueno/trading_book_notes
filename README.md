# 投资读书笔记

投资、交易相关书籍的读书笔记众包项目。


## 开始使用
1. 安装[logseq](https://logseq.com/)
2. 将本项目克隆至本地文件夹
    ``git clone https://github.com/lamueno/trading_book_notes.git``
3. 将相关电子书放置到`./assets/books`目录下（考虑到版权因素，本仓库不会储存相关书籍的电子版。），具体文件名请参考[电子书库与命名规则](#电子书库与命名规则)
4. 在logseq中打开仓库目录
